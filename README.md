\# llama.cpp Launcher



A lightweight Python launcher for llama.cpp built for local LLM workflows and experimentation.



Supports:

\- multiple llama.cpp runtimes (`stable` / `latest`)

\- multiple model variants

\- multimodal (`mmproj`) models

\- MTP/speculative decoding variants

\- runtime-tunable parameters



Features:

\- structured profile configuration

\- validation before launch

\- last-used quick launch

\- command preview

\- profile tags, notes, and stability warnings

\- environment-variable-based paths



Built to make launching and switching between llama.cpp configurations faster and easier.
